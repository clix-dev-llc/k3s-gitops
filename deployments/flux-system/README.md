### Bootstrap Flux V2 toolkit
https://toolkit.fluxcd.io/
https://toolkit.fluxcd.io/guides/installation/
```
curl -s https://toolkit.fluxcd.io/install.sh | sudo bash

# enable completions in ~/.bash_profile
. <(flux completion bash)
```
```
# Verify
flux check --pre
```
